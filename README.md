# aws-ssh-pubkey
### PubKeyAuthentication
```
ssh-keygen
cat ~/.ssh/config
```
### AWS
```
1. go to "key pairs" service
2. upload private key
```

```
1. plugin ssh remote
2. ssh configure file
Host 18.218.149.72
```
